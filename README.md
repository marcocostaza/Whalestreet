# WhaleStreet — Landing page

Landing page del sito **WhaleStreet**.

---

## Cosa c'è in questo progetto

- **index.html** — Pagina principale del sito.
- **docs/** — Guide per lavorare sul progetto (workflow, commit, dove cliccare).
- **CHANGELOG.md** — Registro delle modifiche importanti al sito.

---

## Come aggiornare il sito (in breve)

1. Modifica i file qui in Cursor (o chiedi all'AI).
2. Fai **commit** con un messaggio chiaro (pannello **Source Control** → icona ramo a sinistra).
3. Fai **Push** (Publish Branch / Sync Changes) per inviare su GitHub.
4. Vercel aggiorna il sito in automatico.

## Dove cliccare in Cursor

- **Source Control (Git):** barra laterale sinistra → icona **ramo** (branch).
- **Commit:** in quel pannello, scrivi il messaggio nella casella "Message" in alto e clicca **Commit** (o ⌘⏎).
- **Inviare su GitHub:** stesso pannello → **Publish Branch** o **Sync Changes** o **Push**.

Per i dettagli, apri [docs/COMMIT_GUIDE.md](docs/COMMIT_GUIDE.md).
